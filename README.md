# Welcome My Template with Next.js

## Getting Started

```sh
yarn create next-app -e "https://github.com/nbr41to/my-template-nextjs/tree/simple"
```

## Features

- Next.js
- TypeScript
- Tailwind CSS
- SWR
- Axios

## Cautions

- Base Style：Mantine UI のものを使用
- 余白: Tailwind CSS を使用
- 色: Tailwind CSS を使用
- Media Query: Tailwind CSS を使用
